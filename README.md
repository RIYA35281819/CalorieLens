# 🍎 CalorieLens
AI-powered Food Label Nutrition Extractor

## Features
- Upload food label image
- Extracts nutrition info using OCR
- ML model predicts Healthy/Unhealthy
- Visual nutrition breakdown chart

## Tech Stack
- Python, Streamlit, EasyOCR
- Scikit-learn (Random Forest)
- Pandas, Matplotlib

## How to Run
pip install -r requirements.txt
streamlit run app.py
